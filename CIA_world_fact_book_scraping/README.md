**Python libraries:**
<br>Selenium
<br>Time
<br>Pandas
<br>Country Converter (coco)
<br>
#### This code and resulting dataset is desinged for use with ArcGIS Pro to map the CIA World Factbook entry (link) with the matching country using the ISO3 code.
